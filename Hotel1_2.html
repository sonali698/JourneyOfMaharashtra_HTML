Hotel Gurjas

Book This Hotel

Hotel Description

Hotel Gurjas is a budget hotel which provides decent accomodation and is known for great service from the staff. The famous hotel is basic but clean and there is a restaurant next door which provides delicious and sumptous food. The main thing about this hotel is that it is near the railway station which makes it easily accessible. The rooms are comfortable and hygienic. Altogether it is a good place to stay if you are on a budget trip.

Address:

Hotel Gurjas (Formerly Hotel Oberoi), Near Osmanpura Circle, Opp Bharat Petroleum Petrol Pump, Station Road, Aurangabad- 431005
